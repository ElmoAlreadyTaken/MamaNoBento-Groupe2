
# Web technology blog

## Getting Started

Start Supabase:

```bash
docker compose \
  -f ./supabase/docker-compose.yml \
  -f ./supabase/dev/docker-compose.dev.yml \
  up
```

Run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
